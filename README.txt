	Que veut dire surbet ?? 

Le surebet, de l'anglais « sure » (« sûr ») et « bet » (« pari »), est une méthode de pari qui vise à garantir le gain en jouant sur les différences de cotes entre bookmakers. La méthode ne fonctionne que si ces différences de cotes sont suffisamment importantes. Le surebet n'est donc que rarement possible : en pratique pendant des périodes de quelques minutes.

-> surbet = gain obligatoire si vous jouer sur les deux joueurs.*

****************************************************************************************************************************************

-> Ce programme est une aide pour trouver une situation de surbet sur le Tennis.

Fonction:
- Annonce si situation de "surbet". (avant le pari)
- Donne des indications de methodes de surbet pendant le match. (Avant le pari)
- Mode surbet "live" pendant le match. (Aprés pari, pendant le match)

****************************************************************************************************************************************
							Aide

Mode 1:
Risque = aucun
Gain obligatoire

Ce mode permet de trouver un pari en situation de surbet.
Si aprés avoir rentrer les côtes des deux joueurs, vous êtes en situation de surbet, le programme vous annoncera le pourcentage de gain "Obligatoire" si vous jouez sur les deux joueurs.
	-> Le programme vous calculera ensuite le gain si vous lui donnez la somme que vous voulez investir.
Si vous n'êtes pas en situation de surbet, le programme vous donnera la methode la moins risqué pour être en mode "Surbet" pendant le match.

Ce pari ce fera donc en une seule phase:
1/ paris sur les deux joueurs.

----------------------------------------------------------------------------------------------------------------------------------------

Mode 2:
Quitte le programme en cours

------

Mode 3:
Risque = le moins possible mais le risque de perdre votre mise de départ est présent !!!

Ce mode permet d'être en situation de surbet aprés la mise sur le favori (donc la côte est figé pour vous) pendant le match afin d'assurer d'avoir un gain obligatoire.

Le pari ce déroule donc en deux phase:
1/ pari sur votre favori.
2/ pari sur l'autre joueur pendant le match.

Si vous lancez ce mode pendant le match:
Le programme vous demandera:
- la côte du premier joueur (celle pour qui vous avez parié, figée donc).
- la somme du pari.
- la côte en cours (qui doit être supérieure à la cote donnée pour aucun gain)
	-> Le programme vous annoncera la mise qu'il vous faudra parier sur le second joueur et le pourcentage de gain obligatoire.
